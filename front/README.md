# Frontend para o contrato Registra Animal

Aplicação Web3 para realizar o registro, transferência, compra a venda de animais

## Configurações do deploy do contrato

Utilizou-se a rede de Teste Sepolia do Ethereum.

+ **Endereço do explorador de Blocos:** https://sepolia.etherscan.io
+ **Hash do contrado:** 0x04895fD14B31F4D48A42b552259f23496D009f12

## Criando a aplicação web para trabalhar com o contrato

Acessar a pasta que será a raiz do projeto e executar o comando:  
`npx create-next-app dapp`

## Iniciando a aplicação em modo de desenvolvimento

Dentro da pasta **dapp** execute o comando:  
`npm run dev`  

## Bibliotecas utilizadas

**Web3.js**  
`npm install web3`
